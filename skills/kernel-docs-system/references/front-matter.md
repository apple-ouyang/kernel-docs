# Front Matter

推荐最小模板：

```yaml
---
summary: 一句话说明这篇文档帮助完成什么判断或操作
read_when:
  - 遇到什么任务、决策或排障场景时先读
---
```

约束：

- `summary` 优先写“作用 + 决策对象”，不要只重复标题
- `summary` 不能写成 `TODO`、`待补充`、`占位`
- `read_when` 表示“AI 在什么场景下应该优先读这篇文档”
- `read_when` 要写成任务触发语句，不要只写“修改前”“需要时”“排查时”

推荐写法：

- `summary`：定义缺页异常排查时先看哪些现场与寄存器
- `read_when`：排查 slab 分配异常前
- `read_when`：判断这个问题属于 VFS 还是具体文件系统实现时
